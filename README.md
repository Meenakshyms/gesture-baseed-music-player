# Gesture-Based Music Player

A computer vision based music player that allows users to control music playback using hand gestures captured through a webcam.

## Features
- Play / Pause using hand gestures
- Next / Previous track control
- Volume adjustment using gestures
- Real-time hand tracking

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Pygame / PyAutoGUI

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Run the application  
   `python main.py`

3. Allow camera access when prompted

## Use Case
Provides a touch-free music control system using gesture recognition.
